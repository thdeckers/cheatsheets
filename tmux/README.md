# Tmux

__Prefix: <ctrl>+a__

| command              | description               |
| ---                  | ---                       |
| <prefix>+"           | split horizontal          |
| <prefix>+%           | split vertical            |
| <prefix>+<arrow key> | move to pane <arrow key>  |
| <prefix>+x           | close/kill pane           |
| <prefix>+d           | detach from session       |
| <prefix>+c           | create new window         |
| <prefix>+&           | close window              |
| <prefix>+z           | toggle zoom pane          |
| <prefix>+!           | create pane as new window |
| <prefix>+]           | insert                    |
| <prefix>+[           | copy mode                 |
