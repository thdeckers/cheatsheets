# Bash Cheat-Sheet

## chezmoi

| command                 | effect             |
| ---                     | ---                |
| chezmoi apply           | apply all dotfiles |
| chezmoi edit <filename> | change file        |



## bat

Cat alternative

| command             | effect                        |
| ---                 | ---                           |
| bat -p              | plain mode without decoration |
| bat -H N:M          | highlight line n to m         |
| bat -r N:M          | only show line n to m         |
| bat --paging always | use the pager alway           |
